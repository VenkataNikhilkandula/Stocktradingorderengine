Stock Trading Order Engine
A high-performance Stock Trading Order Engine built using Java + Spring Boot that simulates core stock exchange functionalities such as placing buy/sell orders, order matching, trade execution, stock management, user management, and order book maintenance.
This project is designed with a clean layered architecture and supports scalability, asynchronous processing, API documentation, and secure API access.

📌 Features
User Management
> Create new users
> Retrieve user details
> Manage user accounts
Stock Management
> Add new stocks
> Fetch stock details
> Maintain stock availability
Order Management
> Place Buy Orders
> Place Sell Orders
> Cancel Orders
> Validate order requests
Order Matching Engine
> Automatic order matching
> Price-time priority matching
> Trade execution processing
Trade Management
> Store executed trades
> Fetch trade history
> Track transaction execution
Order Book
> Maintain Buy/Sell order queues
> Real-time order visibility
> Order sorting and matching
Security
> API authentication enabled
> Request authorization support
> Configurable security settings
API Documentation
> Integrated Swagger/OpenAPI
Logging
> Application logging enabled
> Rotating log support
